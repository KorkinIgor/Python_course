Requests — это библиотека, которая создана для быстрой и простой работы с запросами.
------------------------------------------------------------------------------------------
import requests

response = requests.get('https://google.com')

print(response.status_code)  #200

100 -  информационные
200 -  успешные
300 -  перенаправление
400 -  клиентские ошибки
500 -  серверные ошибки
------------------------------------------------------------------------------------------
import requests

response = requests.get('https://api.github.com/')

r = response.json()

print(r['current_user_authorizations_html_url'])  #https://github.com/settings/connections/applications{/client_id}



POST — запрос, это когда мы отправляем какие либо данные на сервер, у нас есть тело запроса, что мы хотим передать серверу.
GET — запрос, это когда мы хотим получить какие либо данные от сервера, но уже без тела запроса.
---------------------------------------------------------------------------------------------------------------------
import requests
import pprint


cookies = {'ffff':'4442'}

response = requests.get('https://httpbin.org/cookies',cookies=cookies)

print(response.text)

Consol: 
{
  "cookies": {
    "ffff": "4442"
  }
}

Cookes — это небольшие файлы данных, которые хранятся на компьютере пользователя и передаются между браузером и сервером при каждом запросе.
