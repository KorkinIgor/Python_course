Списковые включения (list comprehensions) — это удобный и компактный синтаксис для создания списков в Python.
-------------------------------------------------------------------------------------------------------------
1) Создание списка квадратов чисел:

squares = [x ** 2 for x in range(1, 6)]

print(squares)  #[1, 4, 9, 16, 25]
--------------------------------------------------------------------------------------------------------------
2) Фильтрация элементов списка:


squars = [x ** 2 for x in range(1, 11) if x % 2 ==0]

print(squars)  #[4, 16, 36, 64, 100]

------------------------------------------------------------------------------------------------------------------
3)

sqwwears = []

for i in range(6):
    if i % 2 ==0:
        sqwwears.append("число четное")
    else:
        sqwwears.append("число не четное")

print(sqwwears)  #['число четное', 'число не четное', 'число четное', 'число не четное', 'число четное', 'число не четное']


OR


sqwwears = ["число четное" if i % 2 == 0 else "число не четное" for i in range(6)]

print(sqwwears)  #['число четное', 'число не четное', 'число четное', 'число не четное', 'число четное', 'число не четное']
----------------------------------------------------------------------------------------------------------------------------
4) Списочные включения могут быть и со словарем

sqwwears = {x:x**2 for x in range(6)}

print(sqwwears)  #{0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
